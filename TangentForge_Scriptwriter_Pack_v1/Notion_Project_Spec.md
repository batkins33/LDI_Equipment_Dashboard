# Tangent Forge Scriptwriter AI — Project Board Spec

## Project Overview
- **Project Name:** Tangent Forge Scriptwriter AI
- **Owner:** BA / Tangent Forge
- **Goal:** Build a reusable AI “writers’ room” that can generate podcast episodes, story formats, and reflection segments across multiple concepts and seasons.

## Key Objectives
1. Define stable system prompts for Claude and GPT.
2. Establish a Content Matrix with repeatable episode types.
3. Create a baseline season (8–12 episodes) using different formats.
4. Integrate with audio workflow (AI voice + human commentary).
5. Document the writer workflow so it’s easy to reuse and scale.

## Suggested Notion Databases

### 1. Episodes
**Fields:**
- Title
- Status (Idea / Outline / Draft / Scripted / Recorded / Released)
- Format (AI Journey, Bridge, Mad Libs, Story, Tangent Map, Framework, Reflection)
- Topic Depth (Mind-bender, Emotional, Current Event, Surreal, Bridge)
- Vibe (Playful, Introspective, Brainy, Tender, Provocative, Wonder)
- Source (Real conversation / Fresh concept / Listener prompt / Mad Libs)
- Tangent Level (Low / Medium / High)
- Accountability Focus? (Yes/No)
- Notes
- Link to Script (file or page)
- Recording Date
- Release Date

### 2. Content Matrix
**Fields:**
- Format
- Description
- Typical Length
- Emotional Range
- Example Episodes
- Production Complexity (Low/Med/High)
- Notes

### 3. Prompts & System Configs
**Fields:**
- Model (Claude / GPT-5 / GPT-4.1)
- Prompt Type (System / Starter / Template)
- Description
- Prompt Text
- Last Updated
- Linked Episodes

### 4. Ideas & Fragments
**Fields:**
- Idea Title
- Type (Theme / Quote / Scenario / Conflict / Image / Line of Dialogue)
- Rough Notes
- Potential Format
- Priority (1–5)

## Example Kanban Stages
- Inbox / Raw Ideas
- Shortlisted
- Outlining
- Script Drafting
- Ready for Recording
- Recorded
- Edited
- Released

## Recommended Views
- **By Format:** See all AI Journey episodes, all Bridge episodes, etc.
- **By Status:** Track production pipeline.
- **By Vibe:** Balance emotional tones across a season.
- **By Accountability Focus:** Identify episodes that model self‑ownership.

## Recurring Tasks
- Weekly: Generate 3–5 new episode ideas using the Content Matrix.
- Weekly: Move 1–2 ideas into outline or script stage.
- Monthly: Review what formats and vibes are over/under-represented.
- Quarterly: Refresh system prompts with what’s working best.
