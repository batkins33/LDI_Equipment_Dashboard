# MVP Overview — Unified Ops Dashboard (Mock Server)

## Purpose
Validate a unified integration architecture (ACC + Procore + HCSS) before connecting to real production APIs.

## What it proves
- Bind a "unified job" across multiple systems
- Persist mappings (vendors, cost codes)
- Generate a flagship operational report (Daily Trucking Summary)
- Audit changes via an audit log

## Phase 0 scope
- One pilot job
- 2–3 vendors
- 10–20 tickets
- Cost code map stubs
- Report run history
